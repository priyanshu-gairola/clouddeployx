# 🚀 CloudDeployX

A production-style DevOps project demonstrating containerization, Infrastructure as Code (Terraform), CI/CD automation, and Kubernetes orchestration.

The project showcases how a modern cloud-native application moves from source code to deployment using Docker, GitHub Actions, AWS, Terraform, and Kubernetes.

---

## 🚀 Project Overview

CloudDeployX was built to simulate a real-world DevOps workflow instead of a simple demo application.

The project demonstrates:

- Containerizing a Python application using Docker
- Infrastructure provisioning using Terraform
- Automated CI/CD using GitHub Actions
- Docker image management with Amazon ECR
- AWS-based Docker deployment
- Kubernetes deployment with production best practices such as Deployments, Services, Ingress, ConfigMaps, Secrets, Health Probes, Resource Requests & Limits, and Rolling Updates.

> **Note:** Kubernetes was deployed and tested locally using Docker Desktop Kubernetes. AWS deployment and Kubernetes deployment are demonstrated as two separate deployment environments.
